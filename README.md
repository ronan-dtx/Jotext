**Jotext - 按照日期维度记录文字的工具**

**1.主题**<br>
主题可帮助你区分不同类型的记录，类似于“文件夹”，记录是文件夹内的“文件”。

**2.记录**<br>
在主题下按照日期维度记录，每条记录内可以有多条内容。

**3.查询**<br>
可通过「关键词」或「日期」进行搜索

**4.下载**<br>
在指定主题的「记录列表」下载，可将当前主题内的全部记录汇总在一个txt文件下载至本地保存。
在「主题列表」点击下载，可将全部主题的全部记录下载至本地保存。

**5.导入**<br>
在指定主题的「记录列表」点击「上传」，可上传过往下载的txt文件覆盖当前**主题**的所有记录数据。
在「主题列表」点击「上传」，可上传过往下载的txt文件覆盖当前所有记录数据。
